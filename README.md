## 角色生產點數公式計算(有專武=1, 無專武=0)：

### 二星
```
150 + (當前星數 - 原生星數)x21 + 角色突破數x30 + 是否有專武x150 + 專武突破數x30
```

### 三星
```
300 + (當前星數 - 原生星數)x60 + 角色突破數x60 + 是否有專武x300 + 專武突破數x60
```

如公式有誤請告知
